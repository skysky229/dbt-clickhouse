# Jira Automated Modeling - dbt Project

This dbt project is part of the Jira Automated Modeling automation pipeline.

## Structure

- **models/generated**: Contains models automatically generated from Jira Tickets.
- **models/custom**: (Optional) Place for manual custom models.

## Usage

This project is primarily executed by the Spark/Airflow automation, but can be run locally for development.
